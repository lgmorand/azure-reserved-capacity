# Slides

Drop the presentation here.

- Add the deck as `slides.pptx` (and/or an exported `slides.pdf`) in this folder.
- Once added, update the **Slides** card in `../index.html`:
  - remove the `disabled` class and the `soon` pill,
  - turn the `<div class="card disabled">` into `<a class="card" href="slides/slides.pptx">`.

The GitHub Pages workflow publishes everything under `site/`, so the file will be available at
`https://<user>.github.io/<repo>/slides/slides.pptx`.
